Beautiful Soup¶

First steps¶

Beautiful Soup is a Python library for retrieving data from HTML (and other XML) files. We will explore its basic functionality using the UB Math Department web page with graduate student directory as an example:

https://www.buffalo.edu/cas/math/people/grad-directory.html

[1]:

# display the web page in the notebook

                   (function($) {
    'use strict';

    /**
     * The Local Storage key for loading the last recorded Long-Term Alert Banner toggle state.
     */
    var toggleStateRecordKey = 'ubcmsLongTermAlertToggleState';
    setupAlertBannerCollapsible();
    moveAlertBannerBeforePage();

    /**
     * Sets up the collapsible functionality and elements for a Long-Term Alert Banner that is found on the page.
     */
    function setupAlertBannerCollapsible() {
                var $alertBanner = $('.long-term-alert-banner');
        var $content = $alertBanner.find('ul');
        var $toggleButton = addToggleButton($alertBanner);

        $content.attr('aria-label', 'Long-Term Alert Banner Content');
        initToggleState($alertBanner, $toggleButton, $content);
                setupToggleListener($alertBanner, $toggleButton, $content);
    }

    /**
     * Adds a toggle button to a given Long-Term Alert Banner.
     * @param {jQuery} $alertBanner The Long-Term Alert Banner.
     * @return {jQuery} The added toggle button.
     */
    function addToggleButton($alertBanner) {
                var $toggleButton = $(document.createElement('button'));
        $toggleButton.attr('type', 'button');
        $toggleButton.addClass('collapsible-toggle');
        $toggleButton.attr('aria-haspopup', 'true');
        $toggleButton.attr('aria-label', 'Long-Term Alert Banner Toggle Button');
        $toggleButton.attr('id', 'long-term-alert-collapsible-toggle');
        $alertBanner.append($toggleButton);
        return $toggleButton;
    }

    /**
     * Initializes the toggle state for a given Long-Term Alert Banner and its contents.
     * Note that the toggle state will be initialized to any previously saved toggle state.
     * @param {jQuery} $alertBanner The Long-Term Alert Banner.
     * @param {jQuery} $toggleButton The toggle button.
     * @param {jQuery} $content The toggle content.
     */
    function initToggleState($alertBanner, $toggleButton, $content) {
        var toggleStateRecord = JSON.parse(localStorage.getItem(toggleStateRecordKey));
        var expanded = (toggleStateRecord && toggleStateRecord.version === $alertBanner.text())
                ? (toggleStateRecord.expanded === 'true')
            : true; // Defaults to expanded on first load.

        if (expanded) {
                $toggleButton.attr('aria-expanded', 'true');
                $alertBanner.addClass('expanded');
        } else {
                        $content.addClass('mobile-collapsed');
        }
    }

    /**
     * Sets up a toggle button listener for a given Long-Term Alert Banner.
     * @param {jQuery} $alertBanner The Long-Term Alert Banner.
     * @param {jQuery} $toggleButton The toggle button.
     * @param {jQuery} $content The toggle content.
     */
    function setupToggleListener($alertBanner, $toggleButton, $content) {
        $toggleButton.click(function() {
            toggleCollapsible($alertBanner, $toggleButton, $content);
        });
    }

    /**
     * Toggles a given collapsible Long-Term Alert Banner.
     * @param {jQuery} $alertBanner The Long-Term Alert Banner collapsible container.
     * @param {jQuery} $toggleButton The collapsible toggle button.
     * @param {jQuery} $content The collapsible content.
     */
    function toggleCollapsible($alertBanner, $toggleButton, $content) {
        $alertBanner.toggleClass('expanded');
        var expanded = ($alertBanner.hasClass('expanded') ? 'true' : 'false');
        $content.slideToggle('fast');
        $toggleButton.attr('aria-expanded', expanded);
        saveToggleStateForReload($alertBanner, expanded);
    }

    /**
     * Saves the toggle state of a given Long-Term Alert Banner so that it maintains its toggle state on page re-visit.
     * @param {jQuery} $alertBanner The Long-Term Alert Banner to save the toggle state of.
     * @param {boolean} expanded The epanded toggle state of the Long-Term Alert Banner.
     */
    function saveToggleStateForReload($alertBanner, expanded) {
        var toggleStateRecord = {
            version: $alertBanner.text(),
            expanded: expanded
        };
        localStorage.setItem(toggleStateRecordKey, JSON.stringify(toggleStateRecord));
    }

    /**
     * Moves the a Long-Term Alert Banner found within a page header just before the #page container (if one exists).
     */
    function moveAlertBannerBeforePage() {
        var $page = $('#page');
                if ($page.get(0)) {
            var $alertBanner = $('.long-term-alert-banner');
            $alertBanner.detach().insertBefore($page).wrap('<section aria-label="UB Long-Term Alert">');
            $alertBanner.addClass('ensure-z-top');
        }
    }

})(jQuery);

Links¶

<a href="https://www.buffalo.edu">University at Buffalo</a>

Tables¶

Here is an example of an HTML code defining a simple table:

<table>
  <tr>
    <th>Item</th>
    <th>Quantity</th>
    <th>Price</th>
  </tr>
  <tr>
    <td>Tomatoes</td>
    <td>5</td>
    <td>$3</td>
  </tr>
  <tr>
    <td>Apples</td>
    <td>10</td>
    <td>$4.50</td>
  </tr>
</table>

  • The whole table is enclosed in the <table> tag.

  • Each table row is defined by the <tr> tag.

  • Items in the row with table headers are enclosed in the <th> tag.

  • Data in all other rows is specified using the <td> tag.

Here is the address of a page which shows the table produced by the above code:

https://www.mth548.org/_static/table.html

[29]:

table_url = "https://www.mth548.org/_static/table.html"

page = requests.get(table_url).text

for cell in table.find_all('td'):

We are interested in the cell appearing immediately after apples_cell (for quantity), and the one after that (for price). The method find_next() comes useful here. It searches for the first occurrence of a tag of a specified kind after a given tag:

[33]:

apples_cell.find_next('td') # find the first 'td' tag appearing after apples_cell

apples_cell.find_next('td').find_next('td') # find the second 'td' tag appearing after apples_cell
